# Typescript Sample Code for IoT Device

## Prerequisits

Install [Node.js](https://nodejs.org/)

## Prepare and run "Device Simulator"

1. Update Realm App ID in `./src/realm/config.ts`
2. `cd ./device-ts`
3. `npm install`
4. `npm run build`
6. `npm start`
7. Open web console http://localhost:3001
